Welcome to januslinux - Fast and compact Linux distribution which uses musl libc.

What is this?

januslinux is a fast and compact Linux distribution which uses musl libc. This distribution is made from scratch. Its goal is to be optimized and compact. It uses own fork of package manager called "pkgutils" from CRUX. januslinux is oriented for general use, but it is designed for advanced Linux users. januslinux was compiled with a hardened toolchain for better security. januslinux is a rolling distribution, it allows you to get the latest software. Also, januslinux have pretty good hardware support.
