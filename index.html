Introduction

Ataraxia Linux is an independent, multi-platform, general purpose operating system based on the Linux kernel and musl libc. Ataraxia Linux aims to be simple, compact and safe, adhering strictly to the KISS principle. It can be used on desktops, servers and embedded devices. Ataraxia Linux is available for x86, ARM, MIPS, PowerPC64 (big and little endian) and RISC-V. It uses it's own package manager called "neko", which is compatible with CRUX Pkgfiles. Ataraxia Linux provides better security than standard distributions. For example, every package was compiled with PIC, PIE and SSP and the kernel is configured for better security, without compromise. Ataraxia Linux is oriented towards advanced Linux users, who know what they're doing.

If you want to get help or advice, please, check out our IRC, Telegram, Matrix, Reddit and Discord.
