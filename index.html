Welcome to Ataraxia Linux - Fast and compact Linux distribution which uses musl libc.

What is this?

Ataraxia Linux is a fast and compact Linux distribution which uses musl libc. This distribution is made from scratch. Its goal is to be optimized and compact. It uses own fork of package manager called "pkgutils" from CRUX. Ataraxia Linux is oriented for general use, but it is designed for advanced Linux users. Ataraxia Linux was compiled with a hardened toolchain for better security. Ataraxia Linux is a rolling distribution, it allows you to get the latest software. Also, Ataraxia Linux have pretty good hardware support.
